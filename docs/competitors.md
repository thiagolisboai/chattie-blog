# Concorrentes Monitorados

> Lista de domínios monitorados pelo `competitor-monitor.mjs`.
> Formato: `domain | Nome | foco de busca | região (br|intl)`
>
> **Atualizado em:** 2026-04-20

Uso:

- `node scripts/competitor-monitor.mjs`                  — monitorar todos
- `node scripts/competitor-monitor.mjs --local-only`      — só BR
- `node scripts/competitor-monitor.mjs --intl-only`       — só internacionais
- `node scripts/competitor-monitor.mjs --inject-backlog`  — injeta gaps no backlog
